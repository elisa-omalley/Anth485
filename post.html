My Website Creating Process

a short reflection

Posted by Elisa O'Malley on October 3, 2017

The website creating proccess is not something that everyone thinks about. Most of us see a beautiful webpage without really understanding the intensive process that went into making it. I had taken a computer science class a few years ago, where I learned how to use HTML, CSS, and Javascript, but I never kept up with them, and most of my knowledge was lost. Anthropology 485 gave me a fresh start, to learn and relearn how to create a website in a way that reflects my personality and style.

I began my website creating process by first signing up for GitHub and CodeCademy. Through CodeCademy, I was able to go through user friendly tutorials that taught me how to use HTML, CSS, and Javascript. I was then able to use GitHub and StartBootStrap to "fork" a template of my choosing, and begin my own website. The process of learning how to navigate GitHub took some time. I was lucky enough to have my friend Nick come over for a few tutoring sessions, and help me fix some issues that I had with the website. Through Nick, I learned how to use Terminal and Git command to send my code changes directly to my github webpage. I also learned how to use the Google Chrome "inspector" to find specific code within my website.

I designed my website to reflect my personal style and my way of thinking. I also wanted to include images that showed a side of me that my writing couldn't. It was important to me that my website be easy to navigate and easy to find through social media. The website is meant to be a glance into my life at the University of Victoria. In the future I would like to keep up a blog related to my time at the University and the classes that I'm taking. The blog is an outlet for me to share the things I am learning.

All in all, this website creating process opened a lot of doors for me to explore areas of the digital world that I never knew existed. I am excited to continue working on my site and continue learning new things to make it better.

- Elisa
