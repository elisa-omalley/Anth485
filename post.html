My Website Creating Process

a short reflection

Posted by Elisa O'Malley on October 3, 2017

The website creating proccess is not something that everyone thinks about. We see a beautiful web page without really understanding the intensive process that went into making it. I had taken a computer science class a few years ago, where I learned how to use HTML, CSS, and Javascript, but I never kept up with them and most of my knowledge was lost. Anthropology 485 gave me a fresh start, to learn and relearn how to create a website in a way that reflects my personality and style.

I began my website creating process by first signing up for GitHub and CodeCademy
